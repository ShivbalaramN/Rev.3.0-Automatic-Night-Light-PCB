# Rev.3.0-Automatic-Night-Light-PCB
Project Overview:
This project is a simple Automatic Night Lamp circuit designed using KiCad. The lamp automatically turns ON in darkness and turns OFF in daylight using a Light Dependent Resistor (LDR).

Components Used:
 Light Dependent Resistor (LDR)
 BC547 NPN Transistor
 LED
 10kΩ Resistor
 100kΩ Resistor
 Power Supply (5V / 9V)
 
Tools Used:
KiCad for schematic design
KiCad PCB Editor for board layout

Files Included:
 Schematic diagram
 PCB layout
 3D PCB view
 Gerber files for PCB manufacturing
